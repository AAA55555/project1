Полная интеграция с 1С/Oracle/SAP

Мы сделали LegalDoc максимально гибким при интеграции в ИТ ландшафт. Есть готовые модули для работы с 1C, SAP, Oracle e-Business Suite, есть API для подключения ECM- и BPM-систем.

Работа со всеми типами документов

LegalDoc охватывает работу со всеми типами документов, которые используются между компаниями.

Полная интеграция с операторами ЮЗЭДО

Более 10 лет поддерживаем соответствие законодательству и требованиям операторов ЭДО. Автоматически обрабатываем все существующие электронные документы.

Эффекивная работа с КЭП

Подписывайте документы в 1С/SAP/Oracle/web/iOS/Android. Ключ ЭЦП может быть выдан персоналу или хранится в специальном хранилище компании или облака.
